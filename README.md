# Super Bowl Data Analysis

## Overview

This project analyzes historical Super Bowl data to identify trends in team performance, television viewership, and halftime show popularity.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Dataset Files

* super_bowls.csv
* tv_data.csv
* halftime_show_artists.csv

## Analysis Performed

* Team performance comparison
* Historical game statistics analysis
* TV viewership trend analysis
* Halftime show artist popularity analysis
* Data visualization and trend discovery

## Outcome

Built an analytical dashboard to understand historical trends and audience engagement patterns across Super Bowl events.
